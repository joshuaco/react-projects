# Google Translate using Gemini API

This project is a [Google Translate](https://translate.google.com/) clone using Gemini API to made the translation.

# Build with

- React
- TypeScript
- Vite
- Vitest
- ESLint
- Prettier
- Google Gemini API

## Install

```bash
pnpm install
```

## Run

```bash
pnpm run dev
```
